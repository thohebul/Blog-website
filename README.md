A clean, modern, and fully responsive **Blog Website** built using **HTML, CSS, Bootstrap**. This project features a beautiful gallery section, responsive layout, modern UI components, and a simple blog-style landing page.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/thohebul/Blog-website.git
```

Open the project folder

```bash
cd Blog-website
```

Run the project

Simply open **index.html** in your browser.
